# Contributing to Inventory Management System

Thanks for considering a contribution! Here's how to get started.

## Reporting Issues

If you find a bug or have a feature request, please open an issue and include:

- A clear description of the problem or suggestion
- Steps to reproduce (for bugs)
- Your Python version and operating system

## Submitting Changes

1. Fork the repository
2. Create a feature branch
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes and test them locally by running `python inventory.py`
4. Commit your changes with a clear message
   ```bash
   git commit -m "Add: short description of change"
   ```
5. Push to your fork and open a pull request

## Code Style

- Follow [PEP 8](https://peps.python.org/pep-0008/) conventions
- Keep functions small and focused on a single responsibility
- Add comments for any non-obvious logic

## Code of Conduct

Please be respectful and constructive in all interactions within this project.
